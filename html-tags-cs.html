Headings

Headings from 'h1' to 'h6' identify blocks and sections.
Every page must have one 'h1'. Located under <body> tags.

Lists

Unordered list

Unordered lists are marked with bullet points and go in <ul> tags. Each list item goes into the < li > tags.

Types of lists:

  • Unordered lists
  • Ordered lists

Ordered lists

Ordered lists are numbered and go in <ol> tags. Each list item goes in an <li> tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the <a href="link"> attribute. Go to DuckDuckGo to search for more information.

Setting 'target="blank"' will open the link in a new window. See MDN for more details.

Anchor tags can also create links to a new location on the same page. Instead of putting a URL in the "href" attribute, just use the "id" of the element you want to navigate to. This allows you to jump to the top of the page.

Text Emphasis

Italic

The <em> element adds extra emphasis to the text within the opening and closing tags. This shows up as italic in most browsers.

Bold

You can also use the <strong> element to emphasise text. This shows up as bold text in most browsers.
